常用命令记录

Windows 常用命令

dir #(directory) ：列出当前目录下的文件以及文件夹 
md #(make directory)： 创建目录 
rd #(remove directory)：删除目录 
cd #(change directory)：进入指定目录 
cd.. # 退回到上一级目录 
cd\ # 退回到根目录 
del # 删除文件 
exit # 退出 dos 命令行 
cls #(clear screen): 清屏

Anaconda 常用命令

1、Anaconda 升级

升级 Anaconda 需要先升级 conda

conda update conda
conda update anaconda
conda update anaconda-navigator    #update 最新版本的 anaconda-navigator

2、Anaconda 卸载

依次执行以下步骤

1)、打开 Anaconda Prompt，输入以下命令，删除所有配置文件和目录。

conda install anaconda-clean     # 安装 anaconda-clean
anaconda-clean --yes             # 运行 anaconda-clean

2)、手动删除 anaconda 安装目录下的 envs 和 pkgs 文件夹。

3)、在控制面板中，卸载 anaconda 程序。

3、conda 环境使用基本命令

conda update -n base conda        #update 最新版本的 conda
conda create -n xxxx python=3.5   # 创建 python3.5 的 xxxx 虚拟环境 
conda remove -n xxxx --all        # 移除 xxxx 虚拟环境 
conda activate xxxx               # 开启 xxxx 环境 
conda deactivate                  # 关闭环境 
conda env list                    # 显示所有的虚拟环境

4、更新，卸载安装包

conda list               # 查看已经安装的文件包 
conda list  -n xxx       # 指定查看 xxx 虚拟环境下安装的 package
conda update xxx         # 更新 xxx 文件包 
conda uninstall xxx      # 卸载 xxx 文件包

5、清理（conda 瘦身）

onda clean 就可以轻松搞定！第一步：通过 conda clean -p 来删除一些没用的包，这个命令会检查哪些包没有在包缓存中被硬依赖到其他地方，并删除它们。第二步：通过 conda clean -t 可以将 conda 保存下来的 tar 包。

conda clean -p      # 删除从不使用的包 
conda clean -t      # 删除 tar 包 
conda clean -y -all # 删除所有的安装包及 cache

6、国内镜像源

# 添加 Anaconda 的 TUNA 镜像 
conda config --add channels https:#mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
conda config --add channels https:#mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/conda-forge 
conda config --add channels https:#mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/msys2/

# 需要 pytorch 的话，添加 pytorch 镜像 
conda config --add channels https:#mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/pytorch/

# 设置搜索时显示通道地址 
conda config --set show_channel_urls yes

# 恢复成默认下载源 
conda config --remove-key channels

pip 常用命令

1、源切换

pip3 install xxx -i https:#pypi.Python.org/simple/ #pip 命令中使用 -i 参数来指定镜像地址

#Windows 下，你需要在当前对用户目录下（C:\Users\xx\pip，xx 表示当前使用对用户，比如张三）创建一个 pip.ini 在 pip.ini 文件中输入以下内容：
[global]
index-url = https:#pypi.tuna.tsinghua.edu.cn/simple
[install]
trusted-host = pypi.tuna.tsinghua.edu.cn
